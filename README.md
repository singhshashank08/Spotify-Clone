
### Installation


1. Install 'npm'
   ```sh
   npm install
   ```
1. Install Tailwind Css
   ```sh
   npm install -D tailwindcss
   ```
1. Tailwind.css init
   ```sh
   npx tailwindcss init
   ```
2. Run watch scrpits
   ```sh
   npm run watch
   ```
3. Npm start
   ```sh
   npm start
   ```
